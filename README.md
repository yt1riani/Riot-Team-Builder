# Riot Team Builder

## 📌 Overview

A desktop application that helps players quickly create balanced teams for friendly matches by analyzing recent performance data (win rate, KDA, and kill participation) using the Riot Games API.

## ⚙️ Features

* Riot ID player lookup (Name + Tag)
* Recent match performance analysis
* Score-based team balancing
* Automatic team generation

## 🖥️ How to Use

1. Enter player Riot ID
2. Search player data
3. Add players to the list
4. Generate balanced teams
